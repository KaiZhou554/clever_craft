# Clever Craft

English | [简体中文](README_zh.md)

**Clever Craft** is a Minecraft Bedrock Edition addon that adds a collection of logical crafting recipes to the game.

Many items in vanilla Minecraft feel like they should have recipes, but they don't.  
This addon introduces some reasonable crafting options while keeping the gameplay close to vanilla.

![Demo Image](docs/demo.webp)

## Examples

Some recipes added by this addon include:

- Nether Wart Block → Nether Wart
- Magma Block → Magma Cream
- Jungle Leaves → Cocoa Beans
- Bamboo → Paper
- Gravel → Flint (Stonecutter)

In addition, armor trim smithing templates can be crafted using blocks and Amethyst.

## Philosophy

The goal of Clever Craft is **not to make items easier to obtain**, but to add crafting recipes that feel natural and intuitive.

Some things are intentionally not included because they are already easy to obtain.

## Suggestions

## Installation 
1. Go to the Releases page to download the latest version.
2. Import the behavior pack into Minecraft Bedrock Edition.
3. Enable the behavior pack in the world settings.
